# intotheworld

## サイト概要
### サイトテーマ
海外の観光スポットや絶景スポットを投稿できるレビューサイト

### テーマを選んだ理由
海外が好きでよく絶景スポットなど見ることが好きだから

### ターゲットユーザ
海外が好きな人、旅行が好きな人

### 主な利用シーン
海外旅行するとき、癒されたいとき
